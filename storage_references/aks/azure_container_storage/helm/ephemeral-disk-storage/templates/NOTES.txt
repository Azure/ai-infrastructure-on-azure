This configuration uses local NVMe/ephemeral disk storage for high-performance workloads.

To verify the installation:

  kubectl get storageclass {{ .Values.storage.className }}
  kubectl get pvc {{ .Values.storage.pvcName }}

To use this storage in a pod, mount the PVC:

  volumes:
    - name: ephemeral-volume
      persistentVolumeClaim:
        claimName: {{ .Values.storage.pvcName }}

Note: Ephemeral disk storage is ideal for:
- AI/ML training scratch space on NDv5 series VMs
- High IOPS/low latency workloads
- Temporary data processing

Data on ephemeral disks is not persistent across pod restarts or node changes.
